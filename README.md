### Getting Started

What things you need to install the software and how to install them

```
git clone https://github.com/tomterado/Anchor-Clone 
```
```
cd Anchor-Clone
```
```
npm install
```
```
cd ios
pod install
```
```
cd ..
react-native run-ios
```
